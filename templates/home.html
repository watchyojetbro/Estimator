Subjects you'll encounter in your first year at TU Darmstadt

Funktonale und objektorientierte Programmierkonzepte (10CP)

This course introduces major programming paradigms. Students learn object-oriented programming (OOP) with concepts like classes, inheritance, polymorphism, and encapsulation, as well as functional programming with recursion, higher-order functions, immutability, and lambda calculus.

DIGITALTECHNIK (5CP)

This subject covers the basics of digital circuits and systems. Topics usually include number systems, Boolean algebra, logic gates, combinational and sequential circuits, finite state machines, and the design of simple processors. It builds the foundation for understanding hardware and low-level computing.

Rechnerorganisation (5CP)

This subject introduces the internal structure of computers. Students learn about digital logic, instruction sets, CPU architecture, memory hierarchy, and input/output systems. It explains how hardware executes machine instructions and how high-level programs map down to the hardware level.

Algorithmen und Datastrukturen (10CP)

This course covers fundamental techniques for solving computational problems efficiently. Students learn about classical algorithms (sorting, searching, graph algorithms) and data structures (arrays, linked lists, stacks, queues, trees, hash tables, graphs). The focus is on understanding complexity (Big-O notation) and choosing the right data structure for a problem. It’s a cornerstone of computer science.

Aussagen- und Prädikatenlogik (9CP)

This course introduces formal logic, which is fundamental in theoretical computer science and software verification. Students study propositional logic (truth tables, logical connectives, satisfiability) and predicate logic (quantifiers, inference rules, proofs). The goal is to develop rigorous reasoning skills for mathematics, algorithms, and programming language semantics.

Automaten, formale Sprachen und Entscheidbarkeit (5CP)

This is a theoretical computer science course. It focuses on automata theory (finite automata, pushdown automata, Turing machines), formal languages (regular languages, context-free languages, grammars), and computability (decidability, the limits of what can be computed). It provides the mathematical foundations for compilers, programming languages, and understanding the theoretical limits of computation.
